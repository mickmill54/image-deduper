# Code of Conduct

This project follows the
[**Contributor Covenant, version 2.1**](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — opening issues, submitting pull
requests, commenting on either, or contributing in any other way —
you agree to abide by its terms. The full text is at the link above
and is the authoritative source.

## Reporting

If you experience or witness behavior that violates the Code of
Conduct in any of this project's spaces (issues, pull requests,
discussions, code review threads), please report it by opening a
[private contact issue][issues] or by emailing the maintainer.

[issues]: https://github.com/mickmill54/image-deduper/issues/new

The maintainer will review and respond as quickly as practical, and
will keep your report confidential to the extent possible while still
addressing the issue. Reports made in good faith will not result in
retaliation against the reporter.

## Scope

This Code of Conduct applies to all project-managed spaces — including
this repository, its issue tracker, its pull requests, and any
project-affiliated communication channels — and to public spaces when
an individual is representing the project (for example, by using a
project email address, posting via a project social-media account, or
acting as an appointed representative at an online or offline event).

## Enforcement

Enforcement decisions follow the **Enforcement Guidelines** described
in Contributor Covenant 2.1, available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines>.
Outcomes range from a private warning to a permanent ban from project
spaces, depending on the severity and pattern of the behavior.

The maintainer is the final decision-maker on enforcement actions in
this project's spaces.
